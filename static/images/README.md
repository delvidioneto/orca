# Logo do Sistema Orca

## Como adicionar o logo

1. Coloque o arquivo de imagem do logo (orca com tuxedo e batuta) neste diretório com o nome `logo.png`
2. Formatos suportados: PNG, JPG, SVG
3. Tamanho recomendado: 120x120px ou maior (será redimensionado automaticamente)

## Localização do arquivo

O logo deve estar em: `static/images/logo.png`

## Fallback

Se o logo não for encontrado, será exibido um ícone placeholder elegante com as cores do sistema (amarelo e cinza).
